Adaptive Reuse: Office-to-Residential Conversions

Overview

Thesis research focusing on adaptive reuse of Pittsburgh office buildings from 1965-1985, analyzing residential conversion potential with emphasis on daylighting performance.

Research Focus

  • Parametric building iterations using RhinoPython scripts
  • Comprehensive daylighting analysis via ClimateStudio API
  • Machine learning for spatial daylight autonomy prediction
  • Evaluation of 100+ building configurations

Tools & Technologies

Rhino, Grasshopper, RhinoPython, ClimateStudio API, Machine Learning, Python

ASHRAE Design Competition: São Paulo Public Library

Overview

Comprehensive HVAC system design for a three-story public library integrating VRF-DOAS with renewable energy.

  • Operational carbon: 101 kgCO₂/m²

Tools

ClimateStudio, Energy Modeling, ASHRAE Standards, PVWatts

Crossy Road 2: Isometric Arcade Game

Overview

A faithful recreation of the classic arcade game Crossy Road, built from scratch in Python using the cmu_graphics library. The game features an isometric view, endless procedural terrain generation, and progressively difficult obstacles.

Key Features

  • Procedural Generation: Infinite diverse terrain (Grass, Roads, Rivers)
  • Object-Oriented Design: Robust class structure for Lanes, Player, and Obstacles
  • Collision Detection: Precise hitbox management for cars and logs
  • Visual Polish: Custom drawing logic for voxel-style aesthetics

Tools & Technologies

Python, CMU Graphics, Object-Oriented Programming (OOP)
